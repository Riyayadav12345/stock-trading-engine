# 📈 Real-Time Stock Trading Engine

A console-based stock trading simulation engine built entirely in **Pure Java** with no external libraries, demonstrating core DSA concepts.

## 🧠 Core DSA Used
- **PriorityQueue (Max/Min Heap)** → O(log n) Order Book matching
- **HashMap** → Portfolio & symbol lookup O(1)
- **ArrayList** → Trade history
- **ScheduledExecutorService** → Live price simulation (Multithreading)

## ✨ Features
- 📊 Real-time stock price simulation
- 🟢 Limit & Market order support
- 📖 Order Book with Buy/Sell matching engine
- 🔄 Partial order fill handling
- 💼 Portfolio tracker with P&L
- 📋 Trade history log
